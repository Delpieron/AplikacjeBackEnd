Projekt StoreApi, udostepnia usluge do zarzadzania baza danych sklepu z samochodami. Mamy przygotowane bazy danych oraz udostepnione metody do prostego zarzadzania danymi. Api pozwala takze na wysylanie maili. Projekt dziala na platformie .net 3.1 i jest na nim zainstalowany swagger dla ulatwienia pracy nad dalszym rozwojem.
